Example 6 saving a snapshot of a form control's state

Here, we save a snapshot of the original "default" state of the form control. We can use this snapshot to restore the control to that state. The snapshot is simply an observable of the state change events which define the form control.

Because the snapshot is an observable, we can use rxjs operators like `filter()` to prevent some of the state from being synced to a form control.
