Example 6 resetting a form control now behaves as expected (and resets initial values)

Here, when we call reset on the inputControl it goes back to the values it was constructed with.

Note:

  1. Resetting a form control does not eliminate existing errors. Changing the value will cause any internal validator functions to run again, which may eliminate errors, but if any external services have added errors to the form control then those external services will need to clear the error(s).
