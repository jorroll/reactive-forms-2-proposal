Example 7 compatibility: using the new API with components made for the old API

To ease the transition from the old ReactiveFormsModule API to the new API, a compatibility layer should be included.

This example demonstrates using the new FormControl with existing angular material components (which expect an old FormControl). I haven't spent *that* much time on this compatibility directive, so you can probably break it, but hopefully you get the idea.
